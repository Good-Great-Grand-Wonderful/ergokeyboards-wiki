# Ergokeyboards Docs

Documentation for [Good Great Grand Wonderful](https://ergokeyboards.com) keyboards, built with [Docusaurus](https://docusaurus.io/). Started life as the [crosses-v2](https://github.com/Good-Great-Grand-Wonderful/crosses-v2) GitHub wiki.

## Working on the docs

```sh
npm install
npm start        # live-reload dev server
npm run build    # production build into build/
```
