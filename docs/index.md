Build guides, firmware help, and tuning docs for [Good Great Grand Wonderful](https://ergokeyboards.com) keyboards.

## Keyboards

### [Crosses V2](/crosses-v2/)

A split keyboard with dual trackballs, powered by ZMK. Two variants (Crosses and Bridges), three sizes (36, 42, and 54 keys).
